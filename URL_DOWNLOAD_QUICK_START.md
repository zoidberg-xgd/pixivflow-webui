# URL 直接下载 - 快速开始

## 🎯 功能简介

通过输入 Pixiv URL 或作品 ID 直接下载作品,无需配置文件!

---

## 🚀 快速使用

### 1. 打开 URL 下载页面

在侧边栏点击 **"URL 下载"** 菜单项

### 2. 输入 URL 或作品 ID

**支持的格式:**
- `https://www.pixiv.net/artworks/123456`
- `123456` (直接输入作品 ID)

### 3. 开始下载

点击 **"立即下载"** 按钮

### 4. 查看进度

自动跳转到下载任务页面查看进度

---

## 📦 批量下载

### 1. 准备 URL 列表

每行一个 URL 或作品 ID:
```
https://www.pixiv.net/artworks/123456
https://www.pixiv.net/artworks/789012
123456
```

### 2. 粘贴到批量下载框

在右侧 "批量下载" 卡片中粘贴

### 3. 解析验证(可选)

点击 **"解析所有 URL"** 查看哪些 URL 有效

### 4. 开始下载

点击 **"下载全部"** 按钮

---

## ✅ 支持的 URL 格式

| 格式 | 示例 | 类型 |
|------|------|------|
| 标准格式 | `https://www.pixiv.net/artworks/123456` | 插画 |
| 带语言 | `https://www.pixiv.net/en/artworks/123456` | 插画 |
| 旧格式 | `https://www.pixiv.net/member_illust.php?illust_id=123456` | 插画 |
| 小说 | `https://www.pixiv.net/novel/show.php?id=123456` | 小说 |
| 纯 ID | `123456` | 插画 |

## 💡 使用技巧

### 技巧 1: 从浏览器直接复制

在 Pixiv 网站上,直接复制地址栏的 URL,粘贴到 PixivFlow 即可下载。

### 技巧 2: 批量下载收藏

1. 整理一份收藏作品的 URL 列表(Excel/文本文件)
2. 复制所有 URL
3. 粘贴到批量下载框
4. 一键下载所有作品

### 技巧 3: 解析验证

下载前先点击 "解析 URL" 验证:
- ✓ 绿色勾号 = 有效 URL
- ✗ 红色叉号 = 无效 URL

### 技巧 4: 移除无效 URL

批量下载时,可以移除无效的 URL,只下载有效的作品。

## ❓ 常见问题

### Q: 支持哪些网站的 URL?

**A**: 目前只支持 Pixiv (pixiv.net) 的 URL。

### Q: 可以同时下载多个任务吗?

**A**: 不可以。同一时间只能运行一个下载任务。请等待当前任务完成后再启动新任务。

### Q: 下载的文件保存在哪里?

**A**: 保存位置由配置文件中的 `illustrationDirectory` 和 `novelDirectory` 决定。

### Q: 如何查看下载进度?

**A**: 点击下载后会自动跳转到 "下载任务" 页面,可以查看实时进度和日志。

### Q: URL 解析失败怎么办?

**A**: 
1. 检查 URL 格式是否正确
2. 确认是 Pixiv 网站的 URL
3. 尝试直接输入作品 ID

## 🎬 使用示例

### 示例 1: 下载单个插画

```
1. 在 Pixiv 上找到喜欢的插画
2. 复制 URL: https://www.pixiv.net/artworks/123456
3. 粘贴到 PixivFlow 的 "单个下载" 框
4. 点击 "立即下载"
5. 等待下载完成
```

### 示例 2: 批量下载多个作品

```
1. 准备 URL 列表:
   https://www.pixiv.net/artworks/123456
   https://www.pixiv.net/artworks/789012
   https://www.pixiv.net/artworks/345678

2. 粘贴到 "批量下载" 框
3. 点击 "解析所有 URL" 验证
4. 点击 "下载全部 (3)"
5. 查看下载进度
```

### 示例 3: 使用作品 ID

```
1. 只知道作品 ID: 123456
2. 直接输入 "123456"
3. 点击 "立即下载"
4. 系统自动识别为插画并下载
```

## 🔧 故障排除

### 问题 1: "另一个下载任务正在运行"

**解决方案**: 
- 等待当前任务完成
- 或在 "下载任务" 页面停止当前任务

### 问题 2: "无效的 URL"

**解决方案**:
- 检查 URL 格式
- 尝试使用其他格式
- 直接输入作品 ID

### 问题 3: 下载失败

**解决方案**:
- 检查网络连接
- 确认已登录 Pixiv 账号
- 查看日志了解详细错误

## 📱 移动端使用

在移动设备上使用 PixivFlow:

1. 在 Pixiv App 中找到作品
2. 点击分享 → 复制链接
3. 打开 PixivFlow WebUI
4. 粘贴 URL 并下载

## 🎯 最佳实践

### ✅ 推荐做法

- 下载前先解析验证 URL
- 批量下载时移除无效 URL
